# Multistep Form Generator

A browser-based tool for building, previewing, and exporting clean multistep HTML forms — no code, no backend, no dependencies.

## Features
- Add unlimited steps and fields
- Drag to reorder steps and fields
- Live form preview with working navigation
- Exports self-contained HTML/CSS/JS — paste into any project
- Runs entirely in the browser

## Usage
Open `index.html` in any browser. No install or build step required.

## Stack
Vanilla HTML, CSS, and JavaScript only.
